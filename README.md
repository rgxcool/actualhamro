# Hamroyatra

Hamroyatra is a web application built with Spring Boot (Java backend) and React (frontend). It aims to provide a seamless travel planning and management experience.

## About Hamroyatra

Hamroyatra helps users plan, organize, and manage their journeys efficiently. The platform offers features such as itinerary creation, booking management, and travel recommendations.

## Getting Started

### Prerequisites

- Java 17 or higher
- Maven
- Node.js and npm

### Clone the Repository

```bash
git clone https://github.com/Mrcoderv/hamroyatra.git
cd hamroyatra
```

### Backend (Spring Boot)

1. Navigate to the backend directory:
    ```bash
    cd backend
    ```
2. Build the project:
    ```bash
    mvn clean install
    ```
3. Run the application:
    ```bash
    mvn spring-boot:run
    ```

### Frontend (React)

1. Navigate to the frontend directory:
    ```bash
    cd frontend
    ```
2. Install dependencies:
    ```bash
    npm install
    ```
3. Start the development server:
    ```bash
    npm start
    ```

The frontend will typically run on `http://localhost:3000` and the backend on `http://localhost:8080`.

## Notes

- Ensure both backend and frontend servers are running for full functionality.
- Configure database settings in the backend as needed.
